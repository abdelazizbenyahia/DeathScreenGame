# Screen Of Death
By Linh Hoang and Abdelaziz

This is a typing game built on the dragonfly game engine.

###Prequisites
- Burgers
- Coke
- SFML (Yeah, actually you just need this)

Try to destroy all the errors and collect the highest point!!!

Good luck
